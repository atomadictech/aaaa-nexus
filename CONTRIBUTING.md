# Contributing to Atomadic

Thank you for your interest in Atomadic!

## About This Repository

This is a **marketing and integration toolkit** — it contains documentation, examples, and a proof verifier. The API source code is proprietary and not in this repo.

## How You Can Contribute

### Documentation Fixes
Found a typo, broken link, or unclear instruction? Open a PR.

### New Integration Examples
Written an integration with a framework we don't cover yet? We'd love to include it.

See `examples/` for the existing format. Good candidates:
- LangGraph
- Semantic Kernel
- Haystack
- Dify
- n8n

### Bug Reports
If an API endpoint isn't behaving as documented, open an issue using the bug report template.

### Feature Requests
Have an idea for a new endpoint or product? Open an issue using the feature request template.

## What We Don't Accept

- Source code or implementation PRs (this is a docs-only repo)
- Changes to pricing or product claims
- Modifications to the proof verifier that weaken checks
- Content that could expose proprietary IP

## Code of Conduct

Be respectful. Be constructive. Keep it technical.
